INTRUCCIONES DE USO
1. Iniciar la consola de Windows. 
2. Ubicarnos en la carpeta donde se encuentra el programa.
3. Escribir en la consola “javac figurasObjetos.java” (sin las comillas), dar enter.
4. Seguidamente escribir “java figurasObjetos” (sin las comillas), dar enter.
5. Ingresar el número que será el lado del cuadrado y el cubo, dar enter.
6. Se desplegará un pequeño menú el cual puede seleccionar ingresando los números que aparecen a un costado de las opciones (1,2,3,4,5,6).
7. Al seleccionar una opción el programa le devolverá el resultado y automáticamente se desplegará el menú de opciones.
8. Puede cambiar el valor ingresado al inicio del programa dando la opción de cancelar para que el sistema le pida el nuevo numero.
9. Seleccionar la opción de salir para que finalice el programa.
10. Para volver a iniciar el programa repita los pasos 3 y 4
11. FIN.
